Tags

O que são?

Tags em HTML são elementos utilizados para estruturar e formatar o conteúdo de uma página web. Definindo assim, como o navegador deve exibir diferentes partes do seu documento HTML


<h1> até <h6>

Usadas para criar cabeçalhos de diferentes níveis de importância em uma página da web, onde <h1> é o cabeçalho de nível mais alto e <h6> é o de nível mais baixo.


<p>

Tag usada para criar parágrafos. Ela define o início de um novo parágrafo de texto, o que permite que o navegador exiba o conteúdo entre as tags como um bloco de texto separado.


<mark>

Tag usada para destacar ou marcar parte do texto em uma página da web com uma cor de fundo, geralmente amarela, para chamar a atenção do leitor.


<small>

Tag usada para definir texto como sendo de tamanho de fonte menor do que o texto circundante. É comumente usado para indicar informações secundárias ou textos de rodapé em uma página da web.


<i>

Tag usada para deixar texto em itálico.


<u>

Tag usada para sublinhar o texto.


<strong>

Tag usada para deixar texto em negrito.


<ol>

Tag usada para criar listas ordenadas, também conhecidas como listas numeradas.


<ul>

Tag usada para criar listas não ordenadas, também conhecidas como listas de marcadores. O conteúdo dentro dessa tag é organizado em itens de lista que são marcados com símbolos, geralmente pontos ou outros marcadores.


<li>

Tag usada para definir cada item de uma lista


<a>

Tag usada para criar links em uma página da web. Ela permite que os usuários cliquem no texto ou em um elemento de imagem para navegar para outra página da web, um arquivo, um endereço de email, ou para outro recurso online.


<hr>

Tag usada para criar uma linha horizontal ou uma divisória em uma página da web.


<sub>

Tag usada para criar texto subscrito em uma página da web.


<sup>

Tag usada para criar texto sobrescrito em uma página da web.


<blockquote>

Tag usada para criar uma citação longa ou um bloco de texto que deve ser destacado do restante do conteúdo da página.


<font>

Usada anteriormente em HTML para definir o tamanho, a cor e a família da fonte de texto.


<del>

Tag usada para indicar texto que foi deletado ou removido de um documento.


<abbr>

Tag usada para definir uma abreviação ou um acrônimo em uma página da web.
